MinGW é uma versão portada para Microsoft Windows do conjunto de ferramentas GNU.

Este software inclui um conjunto de arquivos cabeçalho para a API do Windows que permite

aos desenvolvedores usar o GCC para criar programas nativos em Windows sem precisar

contar com uma emulação em tempo real de um sistema Unix-like.

gcc teste.c -o teste.exe

gcc Nomedofonte.c -o NomedoExecutavel.exe

Uma das Vantagens de se usar esse compilador nativo e rodar a compilação em linha de comando é poder usar qualquer editor de código ou IDE
